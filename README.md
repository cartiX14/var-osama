## 🐧 Commandes utiles Arch Linux

```bash
# Installer un paquet
sudo pacman -S <package>

# Mettre à jour tout le système
sudo pacman -Syu

# Rechercher un paquet
pacman -Ss <mot-clé>

# Voir l’état d’un service
systemctl status sshd

# Voir les logs récents
journalctl -xe

```
